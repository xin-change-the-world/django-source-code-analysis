# 1 框架和项目目录

本书目的在于django源码的剖析，所以python的安装，django的安装就略过了。

安装完python2.7和django1.6.2以后，并设置好djang-admin.py的环境变量，在命令行中切换到项目目录下，执行django-admin.py startproject mysite 自动创建项目目录。
